# Solus Project Status

## MVP Scope

The current minimum playable target is the material in Epic 1 and Epic 2.

### Epic 1: Core Gameplay Loop

This epic defines the foundation of the game.

1. Mission statement and system identity
2. Character structure and the systems that affect action resolution
3. What parts of a character stay fixed, change between sessions, or shift during play
4. NPC structure and rank tiers
5. The core gameplay loop and action resolution loop
6. Character advancement through XP, skills, support skills, and masteries

### Epic 2: Playtesting Standards

This epic checks whether the MVP can survive table use.

1. Review the Epic 1 material as a playable rules set
2. Run a rough scene or playtest against that material

## What The Repository Already Covers

- A clear mission statement for Solus as heroic gritty realism with system-driven resolution.
- Five core stats: Body, Mind, Social, Magic, and Sanity.
- A 2d10 additive resolution model for combat rolls, resist rolls, and skill checks.
- Skill design built from one fixed primary stat and one chosen secondary stat.
- Rules for changeable and fixed parts of a character sheet.
- NPC ranks from Minion through Boss.
- A core loop of problem, player action, consequence, reward, adjustment, and repeat.
- Advancement tied to XP spending, skill ranks, support skill slots, and mastery unlocks.
- A review checklist for character creation, enemies, and combat handling.

## Major Unfinished Areas

These sections still need design work, cleanup, or full drafting before the project is ready for book layout.

- Combat edge cases, including clearer death handling and what happens at 0 HP.
- Final stack logic for ongoing status effects such as Burn and Chilled.
- Conditions, status effects, curses, environmental effects, lingering injuries, madness, and reputation.
- Weapons, armor tuning, damage resistance caps, and support for broader equipment rules.
- Races, backgrounds, and mastery trees beyond the current outline stage.
- Spellcasting rules and the setting-side lore for magic.
- Money, general resources, equipment, and languages.
- Social mechanics beyond the current degree-of-5 framework.
- Cleanup of duplicated passages and rough-note sections inside the existing story files.

## Current Read On The Repo

This repository already contains enough structure to describe Solus as a playable draft. It does not yet contain a clean, publication-ready rules manuscript. The next layer of work is consolidation: finish the missing systems, resolve contradictions, and turn design notes into book-ready text.

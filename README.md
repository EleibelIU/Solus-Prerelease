# Solus Prerelease

Solus is a tabletop RPG project in active development. This repository holds the working drafts for the rules, the setting, and the playtest structure that will feed two formatted releases: a setting book and a system book built in Homebrewery and GM Binder.

This is the source vault. The material here is still moving. Some sections read like clean draft text, some read like design notes, and some still mark open questions that need a pass before they belong in a finished book.

## What This Repository Contains

- Core system work for Solus, including the mission statement, character framework, NPC structure, action resolution loop, combat notes, and advancement rules.
- Playtest-facing material that checks whether the current rules support character creation, encounter building, and table use.
- Scratch documents that track combat needs, sample character sheets, and unresolved design work.
- Obsidian vault files used to organize and edit the project locally.

## Working Rule For This Repo

Treat the Markdown files in this repository as the current source of truth for Solus. If a section needs cleanup, tighten the draft in the source file first. The formatted book versions should come from this material, not compete with it.
